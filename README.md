Rescue land
===========

Overview
--------

This is an in-development lander-style game for the First ThoughtWorks Recife Game Contest. The goal is to go through the stages controlling your lander to go into caves and rescue hostages.

Where to play
-------------

Access http://tnaires.github.com/rescue_land/ to play the alpha (I said **ALPHA!**) version.

How to play
-----------

Arrow up - boost lander    
Arrow left - shift lander to the left    
Arrow right - shift lander to the right
